Africa: How has income
inequality changed
over time?

The Gini coefficient is a measure of inequality. It can change in response

to policy decisions, national and world events, and economic factors.

This rotated scatter plot displays changes in the coefficient for

any two 5-year periods. Only those countries for which data

are available in both selected periods appear on the plot.

Hover and click on each point on plot for more data.
